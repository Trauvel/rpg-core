# D&D 5e SRD Data

Эта папка предназначена для хранения данных из D&D 5e System Reference Document (SRD).

## Структура данных

В будущем здесь будут размещены JSON файлы с данными из SRD:

- `classes.json` - классы персонажей
- `races.json` - расы персонажей
- `spells.json` - заклинания
- `items.json` - предметы и экипировка
- `features.json` - черты классов и рас
- `backgrounds.json` - предыстории персонажей

## Источники данных

D&D 5e SRD доступен по лицензии Open Gaming License (OGL).

Популярные источники JSON данных:
- [5e-database](https://github.com/5e-bits/5e-database)
- [D&D 5e API](https://www.dnd5eapi.co/)
- [Open5e](https://open5e.com/)

## Использование

Данные из этой папки будут загружаться в справочные таблицы базы данных:
- `DndClass` - классы
- `DndRace` - расы
- И другие справочные таблицы

## Примечание

На данный момент папка подготовлена для будущей интеграции. 
Загрузка данных и миграция на PostgreSQL будут выполнены позже.
